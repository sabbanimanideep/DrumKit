## How to Use ğŸ—ï¸
1. Open the `index.html` file in your browser.
2. Press the following keys on your keyboard to play the drum sounds:
   - **A** (65) - Tom drum
   - **S** (83) - Hi-hat
   - **D** (68) - Ride cymbal
   - **F** (70) - Open hi-hat
   - **G** (71) - Kick drum
   - **H** (72) - Snare drum
   - **J** (74) - Clap sound
   - **K** (75) - Tribal drum
